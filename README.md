<h2>Description</h2>
<p>
	Program for checking difference between two files in the same and different extension.
	You can use it in terminal and as library.
</p>

<hr>

<h2>Usage</h2>

<p>
	<h3>Helper</h3>
	<pre>gendiff -h</pre>
</p>

<p>
	<h3>Check version</h3>
	<pre>gendiff -v</pre>
</p>

<p>
	<h3>Example</h3>
	<pre>gendiff ~/file1.json ~/file2.yaml [-f] [format name]</pre>
</p>
